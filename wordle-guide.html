📖 Wordle Complete Guide

What is Wordle?

Wordle is a daily word guessing game where you have 6 attempts to guess a 5-letter word.

How to Play

  1. Type a 5-letter word
  2. Green = Correct letter, correct position
  3. Yellow = Correct letter, wrong position
  4. Gray = Letter not in word

Tips for Beginners

Start with words containing common letters.
